# Unity-Evaluate
模拟Eval()的效果，传入算式字符串，返回float结果。

### 支持的运算符，所有字母均不区分大小写

| 运算符 | 效果 | 示例 |
| ----------- | :------------ | :------------ |
| + | 加法 | 1 + 2 = 3 |
| - | 减法 | 1 - 2 = -1 |
| * | 乘法 | 2 * 3 = 6 |
| / | 除法 | 3 / 2 = 1.5 |
| () | 括号 | (1 + 2) * 3 = 9 |
| max | 取左右两个数中大的那个 | 1 max 2 = 2 |
| min | 取左右两个数中小的那个 | 1 min 2 = 1 |
